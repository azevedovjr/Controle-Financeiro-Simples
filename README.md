# Controle-Financeiro-Simples

Aplicação de controle financeiro desenvolvida em Python que permite registrar receitas e despesas, visualizar saldo atual e consultar histórico de transações. O sistema funciona através de um menu interativo no terminal e utiliza listas para armazenar os dados.

Tecnologias utilizadas

Python

Lógica de programação

Estruturas de repetição

Manipulação de listas
